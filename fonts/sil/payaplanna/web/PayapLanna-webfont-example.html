Using Payap Lanna webfonts

The page demonstrates how to use the WOFF2 version of Payap Lanna for web pages, using local copies of the .woff2 files in this archive. For more information on this technique - and how to use the normal TrueType version as a web font - see Using SIL Fonts on Web Pages.

The body of this document is styled using an external stylesheet, with Payap Lanna set as the default font, and loaded directly as a WOFF2 file using the @font-face rule. If you're seeing a serif font here then it is working. Otherwise the text will be in a sans-serif or your browser default. This technique is supported in all modern browsers.

To see specifically how these techniques are specified in the CSS and HTML view the source to this page and the accompanying CSS file.

﻿ᨻᩱ​ᩋ᩠ᩅᩯ᩵​ᨠ᩠ᩅᨯ​ᨹᩬᩴ᩵​ᨷ᩶ᩣ᩠ᨶᨾᩮᩨᩬᨦ​

ᩉ᩠ᩃᩢᨦ​ᨧᩣ᩠ᨠ​ᨴᩦ᩵​ᩉ᩠ᨾᩬᩴ​ᩅᩁᩪᨾ᩠ᨶᩯ​ᨾᩣ​ᨹ᩠ᩅᩯ​ᨯᩱ᩶​ᨷᩴ᩵​ᨻᩬᩴ​ᨾ᩠ᨶᩮᩥ​ᨴᩮᩢ᩵ᩣᨯᩱ ᩁᩮᩢᩣ​ᨲ᩠ᨦᩢ᩶ᨧᩱ​ᨠ᩠ᨸᩢ​ᨲ᩠ᨠᩫᩃ᩠ᨦᩫ​ᨠ᩠ᨶᩢ​ᩅ᩵ᩣ​ᨧᩡ​ᩋᩬᨠ​ᩋ᩠ᩅᩯ᩵​ᨹᩬᩴ᩵​ᨷ᩶ᩣ᩠ᨶ​ᨾᩮᩨᩬᨦ ᨲᩣᩴ​ᩉ᩠ᨾᩪ᩵​ᩉ᩠ᨶᩦ᩶ ᨠᩣ᩠ᩁ​ᨴᩦ᩵​ᨧᩡ​ᩁᩪ᩶​ᨠ᩠ᨿ᩵ᩅᨠ᩠ᨸᩢ​ᨤ᩠ᩅᩣ᩠ᨾ​ᨠ᩠ᩅ᩶ᩣ᩠ᨦ​ᩉ᩠ᨿᩱ᩵​ᨡᩬᨦ​ᨻ᩠ᨶᩨ᩶​ᨴᩦ᩵​ᨠ᩠ᨸᩢ​ᩁᩪ᩶​ᩅ᩵ᩣ​ᨾᩦ​ᨤ᩠ᨶᩫ​ᩀᩪ᩵​ᨶ᩠ᨠᩢ​ᩉ᩠ᨿᩬ᩠᩶ᨶ​ᨴᩮᩢ᩵ᩣᨯᩱ​ᨶ᩠ᨶᩢ᩶​ᨣᩴ᩵​ᨸ᩠ᨶᩮ​ᩁᩮᩨᩬ᩵ᨦ​ᨧᩣᩴᨸ᩠ᨶᩮ​ᨡ᩠ᨶᩣ᩠ᨯ ᨪ᩠ᨦᩮᩥ᩵​ᨧᩡ​ᨯᩱ᩶​ᨡᩬᩴ᩶ᨾᩪᩃ​ᨧᩣ᩠ᨠ​ᨠᩣ᩠ᩁ​ᩋᩬᨠ​ᩋ᩠ᩅᩯ᩵​ᨡᩬᨦ​ᨲ᩠ᩅᩫ​ᩁᩮᩢᩣ ᨠ᩠ᨸᩢ​ᨶᩱ​ᩅᩮᩃᩣ​ᨯ᩠ᨿᩅ​ᨠ᩠ᨶᩢ​ᨣᩴ᩵​ᨧᩡ​ᨯᩱ᩶​ᩋᩪ᩶​ᩁᩮᩨᩬ᩵ᨦ​ᨻᩕᨧᩮᩢ᩶ᩣ​ᨴ᩠ᩅ᩠ᨿ ᨠ᩠ᨸᩢ​ᩉ᩠ᨾᩬᩴ​ᩅᩁᩪᨾ᩠ᨶᩯ​ᨧᩡ​ᨯᩱ᩶ ᨾᩦ​ᩅᩮᩃᩣ​ᨿ᩶ᩣ᩠ᨦ​ᨠᩣ᩠ᩁ ᨻᩮᩨᩬ᩵ᩋ​ᨿ᩶ᩣ᩠ᨦ​ᨧᩣ᩠ᨠ​ᨠᩣ᩠ᩁ​ᩁᩢᨠ᩠ᩇᩣ​ᨤ᩠ᨶᩫ​ᨸ᩠ᨶᩮ​ᨻ᩠ᨿᩥᩣ᩠ᨲ ᨠᩣ᩠ᩁ​ᩋᩬᨠ​ᩋ᩠ᩅᩯ᩵​ᨻᩮᩨᩬ᩵ᩋ​ᩉᩣ​ᨡᩬᩴ᩶ᨾᩪᩃ​ᩉ᩠ᨶᩦ᩶​ᨧᩡ​ᨿ᩠ᨿᩮᩡ​ᩉᩨ᩶​ᨤ᩠ᨶᩫ​ᩋ᩠ᨶᩨ᩵᩻​ᩁᩪ᩶​ᨧ᩠ᨠᩢ​ᨦᩣ᩠ᨶ ᨡᩬᨦ​ᩉ᩠ᨾᩬᩴ​ᨴ᩠ᩅ᩠ᨿ ᨸᩮᩢ᩶ᩣᩉ᩠ᨾᩣ᩠ᨿ​ᨴᩦ᩵​ᩁᩮᩢᩣ​ᨲᩬ᩶ᨦ​ᨠᩣ᩠ᩁ​ᨻᩱ​ᩉᩨ᩶​ᨹ᩠ᩅᩯ​ᨤᩨ ᨾᩮᩨᩬᨦ ᩉ᩠ᩃ᩠ᩅᨦᨻᩕᨷᩣ᩠ᨦ ᨪ᩠ᨦᩮᩥ᩵​ᨸ᩠ᨶᩮ​ᨧ᩠ᨦᩢᩉ᩠ᩅᩢᨯ​ᨶ᩠ᨦᩧ᩵​ᨴᩦ᩵​ᩉ᩠ᨿᩱ᩵​ᨴᩦ᩵​ᩈᩩᨯ​ᨡᩬᨦ ᩈ᩠ᨿᩣ᩠ᨾ᩻ ᨠ᩠ᨸᩢ​ᩀᩪ᩵​ᨠ᩠ᩃᩱ​ᨴᩦ᩵​ᩈᩩᨯ​ᨴ᩠ᩅ᩠ᨿ ᨠᩣ᩠ᩁ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ​ᨻᩱ​ᨴᩮᩨᩬ᩵ᩋ​ᩉ᩠ᨶᩦ᩶​ᨶ᩵ᩣ​ᨧᩡ​ᨷᩴ᩵​ᩃᩣᩴᨷᩣ᩠ᨠ​ᩈ᩠ᨠᩢ​ᨴᩮᩢ᩵ᩣᨯᩱ ᨠ᩠ᨸᩢ​ᨶ᩵ᩣ​ᨧᩡ​ᨯᩱ᩶​ᩁ᩠ᨸᩢ​ᨹ᩠ᩃᩫ​ᨴᩦ᩵​ᨶ᩵ᩣ​ᨻᩬᩴ​ᨧᩱ ᩁᩮᩢᩣ ᩅᩣ᩠ᨦᨹ᩠ᨶᩯ​ᩅᩱ᩶​ᩅ᩵ᩣ​ᩈ᩠ᩅ᩵ᩁ​ᩉ᩠ᨿᩱ᩵​ᩁᩮᩢᩣ​ᨧᩡ​ᨡᩦ᩵​ᩁᩮᩨᩬᩋ​ᨻᩱ​ᨴ᩠ᨦᩢ​ᨶᩣᩴ᩶ ᨳ᩠ᨦᩮᩥᨾᩯ᩶ᩅ᩵ᩣ​ᨲᩬᩁ​ᩉ᩠ᨶᩦ᩶​ᨠᩣ᩠ᩁ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ​ᨴ᩠ᨦᩢ​ᨶᩣᩴ᩶​ᨷᩴ᩵​ᩉ᩠ᨾᩰᩡ​ᨣᩴ᩵​ᨲᩣ᩠ᨾ ᨲᩯ᩵​ᨣᩴ᩵​ᨿ᩠ᨦᩢ​ᨯᩦ ᩅ᩵ᩣ​ᨷᩴ᩵​ᨯᩱ᩶​ᨻᩱ​ᩉ᩠ᨿᩮ​ᩃ᩠ᨿᩮᩥ ᨳ᩶ᩣ​ᨷᩴ᩵​ᨻᩱ​ᨲᩬᩁ​ᩉ᩠ᨶᩦ᩶​ᨣᩴ᩵​ᩉ᩠ᨾᩯ​ᨾ᩠ᨶᩮᩥ​ᩀᩪ᩵​ᨴᩦ᩵​ᨧᩡ​ᨾᩦ​ᩋᩰᨠᩣ᩠ᩈ​ᩉ᩠ᨾᩰᩡ​ᩀ᩵ᩣ᩠ᨦᩋᩦ᩶ ᨪ᩠ᨦᩮᩥ᩵​ᨣᩴ᩵​ᨸ᩠ᨶᩮ​ᩀ᩵ᩣ᩠ᨦ​ᩋ᩠ᨶᩢ᩶​ᨴᩯ᩶᩻ ᨤᩨ​ᩉ᩠ᩃᩢᨦ ᨶ᩠ᨸᩢ​ᩅᩮᩃᩣ​ᩉ᩠ᩃᩢᨦ​ᨧᩣ᩠ᨠ​ᨶ᩠ᨶᩢ᩶​ᨳ᩠ᨦᩮᩥ​ᩈ᩠ᨸᩥ​ᩉ᩠ᨠᩫ​ᨸᩦ​ᨴᩦ᩵​ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ​ᨧᩡ​ᨻ᩠ᨠᩥ​ᨻᩱ​ᨴᩦ᩵​ᩉ᩠ᨶᩢ᩶​ᩉ᩠ᨾᩯ​ᨴᩮᩨᩬ᩵ᩋ​

ᨧᩮᩢ᩶ᩣ​ᨻ᩠ᨦᨶ᩠ᨴᩅᩥᨩ᩠ᨩ᩠ᨿᩣᨶᩫᨶ᩠ᨴ᩺​ᨯᩱ᩶​ᩋᩬᨠ​ᩉ᩠ᨶᩢᨦᩈᩨ​ᩈᩣᩴ᩵ᩉ᩠ᩁᩢᨷ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ​ᩉᩨ᩶​ᩁᩮᩢᩣ ᩉᩨ᩶​ᩁᩮᩢᩣ​ᨸ᩠ᨶᩮ​ᩉᩮᩨᩬ᩠ᨾᩁ​ᨴᩪᨲ​ᨡᩬᨦ​ᨴ᩵ᩣ᩠ᨶ ᨴᩦ᩵​ᨧᩡ​ᨯᩱ᩶​ᩁ᩠ᨸᩢ​ᨠᩣ᩠ᩁ​ᨲᩬ᩶ᩁ​ᩁ᩠ᨸᩢ​ᨯ᩠ᩅ᩠ᨿᩰᩫ​ᨷᩴ᩵​ᨲᩬ᩶ᨦ​ᨧ᩵ᩣ᩠ᨿ​ᨦ᩠ᨶᩮᩥ​ᨸ᩠ᨶᩮ​ᨣ᩵ᩣ​ᩋᩡᩉ᩠ᨿᩢᨦ​ᩈ᩠ᨠᩢ​ᩀ᩵ᩣ᩠ᨦ ᨲᩯ᩵​ᨶᩱ​ᨤ᩠ᩅᩣ᩠ᨾ​ᨸ᩠ᨶᩮ​ᨧ᩠ᨦᩕᩥ​ᨴᩯ᩶᩻​ᩓ᩠ᩅ​ᩁᩮᩢᩣ​ᨲᩬ᩶ᨦ​ᨧ᩵ᩣ᩠ᨿ​ᨣᩪ᩵​ᩀ᩵ᩣ᩠ᨦ ᩈᩩᨯ​ᨴᩣ᩠ᨦ ᨶᩱ​ᩉ᩠ᨶᩢᨦᩈᩨ​ᨡ᩠ᨿᩁ​ᨠ᩠ᨿ᩵ᩅᨠ᩠ᨸᩢ​ᩁᩮᩢᩣ​ᩅ᩵ᩣ ᩁᩮᩢᩣ​ᨸ᩠ᨶᩮ​ᨣᩕᩪ​ᩈᩬᩁ​ᩆᩣᩈ᩠ᨶᩣ ᨠ᩠ᨸᩢ​ᨸ᩠ᨶᩮ​ᩉ᩠ᨾᩬᩴ​ᩁᩢᨠ᩠ᩇᩣ​ᨤ᩠ᨶᩫ​ᨧ᩠ᨸᩮᩢᨸ᩠ᩅ᩠ᨿ᩵ ᨿ᩠ᨿᩮᩡ​ᩀ᩵ᩣ᩠ᨦᩋᩦ᩶​ᨸ᩠ᨶᩮ ᩉᩮᩨᩬ᩠ᨾᩁ ᨷᩬᨠ​ᩉᩨ᩶​ᨤ᩠ᨶᩫ​ᨧ᩠ᨸᩮᩢ ᨤ᩠ᨶᩫ​ᨸ᩠ᩅ᩠ᨿ᩵​ᨻᩣ​ᨠ᩠ᨶᩢ​ᨾᩣ​ᩉᩣ​ᩁᩮᩢᩣ ᨻᩮᩨᩬ᩵ᩋ​ᩉᩨ᩶​ᩁᩮᩢᩣ​ᩁᩢᨠ᩠ᩇᩣ ᩉ᩠ᩃᩣ᩠ᨿ​ᨲᩴ᩵​ᩉ᩠ᩃᩣ᩠ᨿ​ᨴᩮᩨᩬ᩵ᩋ​ᨴᩦ᩵​ᩁᩮᩢᩣ​ᨳᩪᨠ​ᩉ᩠ᨾᩪ᩵​ᨡᩮᩢᩣ​ᩅ᩵ᩣ​ᩉᩨ᩶​ᩅ᩵ᩣ ᩁᩮᩢᩣ​ᩀᩪ᩵​ᨷᩴ᩵​ᨾ᩠ᨶᩮᩥ​ᨻᩬᩴ​ᨴᩦ᩵​ᨧᩡ​ᨿ᩠ᨿᩮᩡ​ᨲᩣ᩠ᨾ​ᨴᩦ᩵​ᨡ᩠ᨿᩁ​ᨶᩱ​ᩉ᩠ᨶᩢᨦᩈᩨ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ​ᨡᩬᨦ​ᨧᩮᩢ᩶ᩣ​ᨾᩮᩨᩬᨦ ᨪ᩠ᨦᩮᩥ᩵​ᩁᩮᩢᩣ​ᨣᩴ᩵​ᨯᩱ᩶​ᨩᩦ᩶​ᨺ᩠ᨶᩫ​ᨾ᩠ᨯᩨ​ᨴ᩠ᨠᩧ​ᨷ᩠ᨶᩫ​ᨼ᩶ᩣ ᨠ᩠ᨸᩢ​ᨩᩦ᩶​ᩉ᩠ᨶᩫ​ᨴᩣ᩠ᨦ ᨴ᩠ᨿᩅ​ᨡᩬᨦ​ᩁᩮᩢᩣ​ᨿ᩠ᨦᩢ​ᩉ᩠ᨾᩯ​ᨠ᩠ᩃᩱ​

ᨶᩱ​ᩉ᩠ᨾᩪ᩵​ᩁᩮᩢᩣ​ᨾᩦ​ᩉ᩠ᩃᩣ᩠ᨿ​ᨤ᩠ᨶᩫ​ᨤᩨ ᩉ᩠ᨾᩬᩴ​ᩅᩁᩪᨾ᩠ᨶᩯ ᨠ᩠ᨸᩢ​ᨲ᩠ᩅᩫ​ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ ᨤ᩠ᨶᩫ​ᨿ᩠ᨿᩮᩡ​ᨠ᩠ᨶᩥ ᨤ᩠ᨶᩫ​ᩁ᩠ᨸᩢ​ᨩᩱ᩶​ᩈ᩠ᩅ᩵ᩁ​ᨲ᩠ᩅᩫ ᨠ᩠ᨸᩢ​ᨤ᩠ᨶᩫ​ᨷ᩠ᨠᩯ​ᨡᩬᨦ ᩉ᩠ᨾᩯ​ᨸ᩠ᨯᩯ​ᨤ᩠ᨶᩫ ᨾᩦ​ᨤ᩠ᨶᩫ​ᨶ᩠ᨦᩧ᩵​ᨴᩦ᩵​ᨡᩮᩢ᩶ᩣ​ᨾᩣ​ᩁ᩠ᨸᩢ​ᨩᩮᩨᩬ᩵ᩋ​ᨻᩕᨧᩮᩢ᩶ᩣ​ᨠ᩠ᨸᩢ​ᩁ᩠ᨸᩢ​ᩆ᩠ᩃᩦ​ᩓ᩠ᩅ​ᨯᩱ᩶​ᨷᩴ᩵​ᨻᩬᩴ​ᨾ᩠ᨶᩮᩥ​ᨾᩣ​ᨸ᩠ᨶᩮ​ᨤ᩠ᨶᩫ​ᨩ᩠ᩅ᩠ᨿ᩵​ᩁᩮᩢᩣ​ᨷᩕᨠᩣ᩠ᩆ ᨩ᩠ᩅ᩵ᨦ​ᨣᩮᩢ᩶ᩣᩉ᩠ᩅᩫᨴᩦ​ᨴᩦ᩵​ᩁᩮᩢᩣ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ​ᩁᩮᩢᩣ​ᨲᩬ᩶ᨦ​ᨩᩱ᩶​ᨩ᩶ᩣ᩠ᨦ ᨤᩨ​ᨧᩣ᩠ᨠ​ᨩ᩠ᨿᨦᩉ᩠ᨾᩱ᩵​ᨳ᩠ᨦᩮᩥ​ᨩ᩠ᨿᨦᩁᩣ᩠ᨿ ᨪ᩠ᨦᩮᩥ᩵​ᩁᩮᩢᩣ​ᩉᩣ​ᨩ᩶ᩣ᩠ᨦ​ᨯᩱ᩶​ᨦ᩵ᩣ᩠ᨿ ᩁᩮᩢᩣ​ᨦ᩠ᨯᩨ ᨡ᩠ᨶᩣ᩠ᨯ​ᨴᩦ᩵​ᨩ᩶ᩣ᩠ᨦ​ᩁᩣᨣᩤ​ᨳᩪᨠ ᩋᩣ᩠ᨧᨧᩡ​ᨿᩬ᩶ᩁ​ᩅ᩵ᩣ​ᨧᩮᩢ᩶ᩣ​ᨡᩬᨦ​ᨩ᩶ᩣ᩠ᨦ​ᨲᩬ᩶ᨦᨠᩣ᩠ᩁ​ᨻᩣ​ᨩ᩶ᩣ᩠ᨦ​ᩉ᩠ᨶᩦ​ᩁ᩠ᨣᩰᩫ​ᩉ᩵ᩣ​ᨴᩦ᩵​ᨠ᩠ᨯᩮᩥ​ᨡ᩠ᨶᩧ᩶ ᩁᩮᩢᩣ​ᩋᩬᨠ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ​ᩅ᩠ᨶᩢ​ᨴᩦ᩵ ᩈ᩠ᨸᩥ​ᩉ᩶ᩣ ᨯᩮᩨᩬᩁ​ᨾᩮᩇᩣ​ᨿ᩠ᨶᩫ ᨻᩩᨴ᩠ᨵᩆᩢᨠᩁᩣ᩠ᨩ ᪂᪄᪁᪅ ᩉ᩠ᩃᩢᨦᨧᩣ᩠ᨠ​ᨠ᩠ᨯᩮᩥ​ᩃ᩠ᨾᩫ​ᨺ᩠ᨶᩫ​ᩉ᩠ᨿᩱ᩵​ᨲᩬᩁ​ᩉ᩠ᩃᩡᨯᩪ​ᨺ᩠ᨶᩫ ᨴᩮᩨᩬ᩵ᩋ​ᩉ᩠ᨶᩦ᩶​ᨸ᩠ᨶᩮ​ᨠᩣ᩠ᩁ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ ᨴᩮᩨᩬ᩵ᩋ​ᨴᩦ᩵​ᨶ᩠ᨦᩧ᩵​ᨡᩬᨦ​ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ ᨲᩣ᩠ᨾ​ᩈ᩠ᨶᩮ᩶ᨴᩣ᩠ᨦ​ᨻᩱ​ᨩ᩠ᨿᨦᩁᩣ᩠ᨿ​ᨪ᩠ᨦᩮᩥ᩵​ᨲᩬᩴ᩵​ᨻᩱ​ᨻᩣ᩠ᨿᩉ᩠ᨶ᩶ᩣ​ᨧᩡ​ᨸ᩠ᨶᩮ​ᩈ᩠ᨶᩮ᩶ᨴᩣ᩠ᨦ​ᨴᩦ᩵​ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ​ᨣ᩠ᨿᩮᩥ​ᩉᩪ​ᨲᩣ ᨣ᩠ᨿᩮᩥ​ᨲᩣ​ᨶ᩠ᨠᩢ​ᨡ᩠ᨶᩧ᩶ ᩉ᩠ᩃᩢᨦᨧᩣ᩠ᨠ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ​ᨾᩣ​ᨲᩣ᩠ᨾ​ᨻ᩠ᨶᩨ᩶​ᨴᩦ᩵​ᨻ᩠ᨿᨦ​ᨡᩬᨦ​ᨩ᩠ᨿᨦᩉ᩠ᨾᩱ᩵​ᩉ᩠ᨶᩫᨴᩣ᩠ᨦ​ᨣᩴ᩵​ᨣᩬ᩠᩵ᨿ᩻​ᩈᩪᨦ​ᨡ᩠ᨶᩧ᩶ ᨧ᩠ᨶᩫ​ᨳ᩠ᨦᩮᩥ​ᩁᩬ᩵ᨾ​ᨯᩬ᩠ᨿ ᨴᩦ᩵​ᨸ᩠ᨶᩮ​ᨴᩦ᩵​ᨠ᩠ᨯᩮᩥ​ᨶᩣᩴ᩶ ᨾᩯ᩵​ᨣ᩠ᩅᨦ ᩉ᩠ᩅ᩵ᩣ᩠ᨦ​ᨴᩣ᩠ᨦ​ᩁᩮᩢᩣ​ᨲᩬ᩶ᨦ​ᨡ᩶ᩣ᩠ᨾ​ᨶᩣᩴ᩶​ᩈᩣ᩠ᨿ​ᩉ᩠ᨶᩦ᩶​ᨳ᩠ᨦᩮᩥ​ᩈᩦ᩵​ᩈ᩠ᨸᩥ​ᨠᩮᩢ᩶ᩣ​ᨴᩮᩨᩬ᩵ᩋ ᨠᩬ᩵ᩁ​ᨴᩦ᩵​ᨧᩡ​ᨡ᩠ᨶᩧ᩶​ᨹ᩠ᨯᩫ​ᨸᩣ᩠ᨿ​ᨯᩬ᩠ᨿ​ᨴᩦ᩵​ᩈᩪᨦ​ᨳ᩠ᨦᩮᩥ​ᩈᩣ᩠ᨾ​ᨻ᩠ᨶᩢ​ᨼᩩᨲ ᩉᩮᩨᩬ᩠ᨶᩋ​ᨶᩣᩴ᩶​ᨴᩃᩮ ᨸ᩠ᨶᩮ​ᩈ᩠ᨶᩢᨸ᩠ᨶᩢᨶᩣᩴ᩶​ᨡᩬᨦ​ᨶᩣᩴ᩶​ᨾᩯ᩵​ᨻ᩠ᨦᩥ​ᨠ᩠ᨸᩢ​ᨶᩣᩴ᩶​ᨾᩯ᩵​ᨡᩬᨦ ᩈᩣ᩠ᨿᨶᩣᩴ᩶​ᨧᩡ​ᩉ᩠ᩃᩱ​ᨻᩱ​ᨴᩦ᩵​ᨾᩯ᩵​ᨶᩣᩴ᩶​ᨠ᩠ᨠᩫ​ᨴᩦ᩵​ᨩ᩠ᨿᨦᩁᩣ᩠ᨿ ᨧᩮᩢ᩶ᩣ​ᨡᩬᨦ​

ᨩ᩶ᩣ᩠ᨦ​ᨴᩦ᩵​ᨾᩣ​ᨠ᩠ᨸᩢ​ᩁᩮᩢᩣ​ᨡᩣ᩠ᨦ​ᨩ᩶ᩣ᩠ᨦ​ᨡᩬᨦ​ᨻ᩠ᨶᩮᩥ᩵​ᩃᩣᩴ᩶​ᨻᩱ ᨿ᩠ᨿᩮᩡ​ᩉᩨ᩶​ᩁᩮᩢᩣ​ᨩᩱ᩶​ᩅᩮᩃᩣ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ​ᨳ᩠ᨦᩮᩥ​ᩈ᩠ᨸᩥ​ᩅ᩠ᨶᩢ ᨠᩣ᩠ᩁ​ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ ᨴᩮᩨᩬ᩵ᩋ​ᩉ᩠ᨶᩦ᩶ ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ​ᨴ᩠ᨿᩅ​ᨲ᩠ᨶᩦ​ᨠᩮᩨᩬᨷ​ᩈᩩᨯ​ᨴᩣ᩠ᨦ ᨪ᩠ᨦᩮᩥ᩵​ᨲᩬᩁ​ᨾᩮᩨᩬ᩵ᩋ​ᩃᩪᩁ​ᨾᩮᩨᩬ᩵ᩋ​ᩁᩮᩢᩣ ᨩᩱ᩶​ᨩ᩶ᩣ᩠ᨦ​ᨡᩬᨦ​ᩁᩮᩢᩣ ᩁᩮᩢᩣ​ᨩᩱ᩶​ᩅᩮᩃᩣ​ᩉ᩠ᨠᩫ​ᩅ᩠ᨶᩢ​ᨴᩮᩢ᩵ᩣᩋ᩠ᨶᩢ᩶​

ᨴᩦ᩵​ᨩ᩠ᨿᨦᩁᩣ᩠ᨿ​ᨹᩪ᩶ᩅ᩵ᩣᩁᩣᨩᨠᩣ᩠ᩁ​ᨲᩬ᩶ᩁ​ᩁ᩠ᨸᩢ​ᩁᩮᩢᩣ​ᩀ᩵ᩣ᩠ᨦ​ᨲ᩠ᨾᩮᩋ᩠ᨠᩫᨲ᩠ᨾᩮᨧᩱ ᨠ᩠ᨸᩢ​ᩁ᩠ᨸᩢ​ᨼ᩠ᨦᩢ​ᩁᩮᩨᩬ᩵ᨦ​ᨡᩬᨦ​ᨻᩕᨧᩮᩢ᩶ᩣ​ᨴ᩠ᩅ᩠ᨿ ᨪ᩠ᨦᩮᩥ᩵​ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ ᨩᩮᩨᩬ᩵ᩋ​ᩅ᩵ᩣ​ᨡᩮᩢᩣ​ᨾᩦ​ᨤ᩠ᩅᩣ᩠ᨾ​ᨩᩮᩨᩬ᩵ᩋ​ᨴᩯ᩶ ᨪ᩠ᨦᩮᩥ᩵​ᩁᩮᩢᩣ​ᨧᩡ​ᨯᩱ᩶​ᩉ᩠ᨶᩢ​ᨠ᩠ᨶᩢ​ᨲᩬᩴ᩵​ᨻᩱ ᨧᩣ᩠ᨠ​ᨴᩦ᩵​ᩉ᩠ᨶᩢ᩶​ᩁᩮᩢᩣ​ᨶ᩠ᨦᩢ᩵​ᩁᩮᩨᩬᩋ​ᨻᩱ​ᨲᩣ᩠ᨾ​ᨶᩣᩴ᩶​ᨠ᩠ᨠᩫ​ᨴᩦ᩵​ᩉ᩠ᩃᩱ​ᩃ᩠ᨦᩫ​ᨶᩣᩴ᩶​ᨡᩬᨦ ᨶᩱ​ᩅ᩠ᨶᩢ​ᩋᩣᨴᩥᨲ᩠ᨿ᩺​ᩁᩮᩢᩣ​ᩀ᩶ᩣ᩠ᨦ​ᩁᩮᩨᩬᩋ​ᨴᩦ᩵​ᨠᩰᩬᩡ​ᨪᩣ᩠ᨿ​ᨠ᩠ᩃᩣ᩠ᨦ​ᨶᩣᩴ᩶​ᩁᩮᩢᩣ​ᩉ᩠ᨶᩢ​ᨲ᩠ᨶᩦ​ᩈᩮᩨᩬᩋ​ᨣ᩠ᨦᩕᩰ᩵​ᨸ᩠ᨶᩮ​ᩉ᩠ᨾᩪ᩵​

ᨷᩴ᩵​ᨾ᩠ᨶᩮᩥ​ᩈ᩠ᨠᩢ​ᨴᩮᩢ᩵ᩣᨯᩱ​ᩁᩮᩢᩣ​ᨣᩴ᩵​ᨾᩣ​ᨹ᩠ᩅᩯ​ᨸᩣ᩠ᨠ​ᨶᩣᩴ᩶​ᨡᩬᨦ ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ​ᨡᩬᩴ​ᨿ᩠ᨠᩫ​ᨦᩣ᩠ᨶ​ᨡ᩠ᨿᩁ​ᨡᩬᨦ ᩋ᩠ᨩᩮ ᩅᩬᩴᨪ᩠ᨦᩥᨴ᩠ᨶᩢ ᩈᨾ᩠ᨴᩥ ᨶᩱ​ᩉ᩠ᨶᩢᨦᩈᩨ ᩉ᩶ᩣ​ᨸᩦ​ᨶᩱ​ᩈ᩠ᨿᩣ᩠ᨾ᩻ ᨾᩣ​ᩃ᩠ᨦᩫ​ᩅᩱ᩶​ᨸ᩠ᨶᩮ​ᨷᩣ᩠ᨦ​ᩈ᩠ᩅ᩵ᩁ ᨤ᩠ᩅᩣ᩠ᨾ​ᩅ᩵ᩣ​

ᨾᩦ​ᨤ᩠ᨶᩫ​ᨧᩣᩴᨶ᩠ᩅᩁ​ᩉ᩠ᨿᩬ᩠᩶ᨶ​ᨴᩦ᩵​ᩉ᩠ᨶᩢ​ᨶᩣᩴ᩶​ᨡᩬᨦ​ᩓ᩠ᩅ​ᨧᩡ​ᨷᩴ᩵​ᩁᩪ᩶​ᩈ᩠ᨠᩧ​ᨲ᩠ᨶᩨ᩵ᨲᩣᨲ᩠ᨶᩨ᩵ᨧᩱ ᩃᩣᩴᨶᩣᩴ᩶​ᨸ᩠ᨶᩮ​ᩁᩬ᩵ᨦ​ᩃ᩠ᨠᩮᩥᩢ​ᨸ᩠ᨶᩮ​ᩈᩣ᩠ᨿ​ᨿᩣ᩠ᩅ​ᩉ᩠ᩃᩱ​ᨠᩣ᩠ᨿ​ᨾᩮᩨᩬᨦ​ᨲ᩵ᩣ᩠ᨦ᩻ ᨧ᩠ᨶᩫ​ᨶ᩠ᨸᩢ​ᨷᩴ᩵​ᨯᩱ᩶​ᩋᩬᨠ​ᨻᩱ​ᩈᩪ᩵​ᨴᩃᩮ ᨾ᩠ᨶᩢ​ᨤ᩠ᨦᩫ​ᨯᩱ᩶​ᨻ᩠ᨸᩫ​ᨸᩡ​ᨠ᩠ᨸᩢ​ᨯᩱ᩶​ᩁᩮᩨᩬ᩵ᨦᩁᩣ᩠ᩅ​ᨲ᩵ᩣ᩠ᨦ᩻​ᨶ᩠ᨠᩢ​ᨡ᩠ᨶᩣ᩠ᨯ ᨳ᩶ᩣ​ᨾ᩠ᨶᩢ​ᩃᩮᩢ᩵ᩣ​ᨯᩱ᩶ ᨴᩦ᩵​ᩉ᩠ᨶᩦ᩶​ᨶᩣᩴ᩶​ᩉ᩠ᩃᩱ​ᨷᩴ᩵​ᨻᩬᩴ​ᩁ᩠ᨦᩯ ᨲᩯ᩵​ᨾᩮᩨᩬ᩵ᩋ​ᨾᩦ​ᩋᩡᩉ᩠ᨿᩢᨦ​ᨾᩣ​ᨠ᩠ᨯᩮᩥᩢ​ᨾ᩠ᨶᩢ​ᨾ᩠ᨶᩢ​ᨧᩡ​ᨾᩦ​ᩈ᩠ᨿᨦ​ᨯ᩠ᨦᩢ​ᩉᩮᩨᩬ᩠ᨾᩁ​ᨼ᩶ᩣ​ᩁᩬ᩶ᨦ​

ᨴ᩠ᨯᩢ​ᨠᩮᩨᩬᨷ​ᨳ᩠ᨦᩮᩥ​ᨸᩣ᩠ᨠ​ᨶᩣᩴ᩶ᨠ᩠ᨠᩫ​ᨾᩦ​ᨾᩮᩨᩬᨦ​ᨩ᩠ᨿᨦ​ᩈ᩠ᨶᩯ ᨸ᩠ᨶᩮ​ᨾᩮᩨᩬᨦ​ᩁ᩶ᩣ᩠ᨦ​ᨷ᩶ᩣ᩠ᨶ​ᨾᩮᩨᩬᨦ​ᩉ᩠ᨠᩢ​ᩉ᩠ᨿᩮ᩵ᨷᩮᩥᩬᩡᩉ᩠ᨿᩮ᩵ᨷᩮᩥᩬ ᨴᩦ​ᨲᩬᩁ​ᨲᨠᩬ᩵ᩁ ᨸ᩠ᨶᩮ​ᨾᩮᩨᩬᨦ​ᩉ᩠ᨿᩱ᩵​ᩉᩮᩨᩬ᩠ᩃᩋ​ᨻ᩠ᨶᩮᩥ᩵​ᨠᩣᩴᨻ᩠ᨦᩯ​ᨾᩮᩨᩬᨦ​ᩉ᩠ᨠᩢ​ᩃ᩠ᨦᩫ​ᨹᩬ᩵​ᩉ᩠ᨶᩢ​ᨸ᩠ᨶᩮ​ᩈᩦ᩵ᩉ᩠ᩃ᩠ᨿ᩵ᨾ​ᩀᩪ᩵​ᩁ᩠ᨾᩥ​ᨲᩣ​ᨺ᩠ᨦᩢ᩵​ᨿᩣ᩠ᩅ​ᨠᩮᩨᩬᨷ​ᩈᩣ᩠ᨾ​ᩃᩰ ᨾᩮᩨᩬ᩵ᩋ​ᨧ᩠ᨯᩮᩢ​ᩈ᩠ᨸᩥ​ᨸᩦ ᨠᩬ᩵ᩁ​ᨴᩦ᩵​ᩁᩮᩢᩣ​ᨧᩡ​ᨾᩣ​ᨹ᩠ᩅᩯ​ᨾᩦ​ᨠᩬᨦᨴ᩠ᨸᩢ​ᨡᩬᨦ​ᩈ᩠ᨿᩣ᩠ᨾ᩻​ᨠ᩠ᨸᩢ​ᩃᩣ᩠ᩅ​ᨯᩱ᩶​ᨡᩮᩢ᩶ᩣ​ᨾᩣ​ᨹᩮᩢᩣ​ᨾᩮᩨᩬᨦ ᩓ᩠ᩅ​ᨲᩬ᩶ᩁ​ᨤ᩠ᨶᩫ​ᨻᩱ​ᨸ᩠ᨶᩮ​ᨡᩦ᩶ᨡ᩶ᩣ ᨲᩬᩁ​ᩉ᩠ᨶᩦ᩶​ᨴ᩠ᨦᩧ​ᨶᩱ​ᨾᩮᩨᩬᨦ​ᨶᩬᨠ​ᨾᩮᩨᩬᨦ ᩅ᩠ᨯᩢ ᩅᩥᩉᩣ᩠ᩁ ᨠᩣᩴᨻ᩠ᨦᩯ ᨳᩪᨠ​ᩃᩡ​ᨡ᩠ᩅ᩶ᩣ᩠ᨦ​ᩉᩨ᩶​ᩈ᩠ᨿᩮᩉᩣ᩠ᨿ​ᨸ᩠ᨶᩮ​ᩁ᩶ᩣ᩠ᨦ ᨾᩣ​ᨸ᩠ᨶᩮ​ᩅᩮᩃᩣ​ᩈᩣ᩠ᨾ​ᩈ᩠ᨸᩥ​ᩈᩦ᩵​ᩈ᩠ᨸᩥ​ᨸᩦ​ᨾᩣ​ᩓ᩠ᩅ ᨴᩦ᩵​ᩉ᩠ᨶᩢ​ᨾᩦ​ᨻᩕᨻᩩᨴ᩠ᨵᩁᩪᨷ​ᨧᩣᩴᨶ᩠ᩅᩁ​ᨶ᩠ᨠᩢ​ᨶ᩠ᨸᩢ​ᨧ᩠ᨶᩫ​ᨻᩬᩴ​ᨷᩴ᩵​ᨻᩯ᩶ ᨿ᩠ᨿᩮᩡ​ᩉᩨ᩶​ᨩᩮᩨᩬ᩵ᩋ​ᩅ᩵ᩣ​ᨩᩣ᩠ᩅᨷ᩶ᩣ᩠ᨶ​ᨩᩣ᩠ᩅᨾᩮᩨᩬᨦ​ᨴᩦ᩵​ᩉ᩠ᨶᩦ᩶​ᨸ᩠ᨶᩮ​ᨤ᩠ᨶᩫ​ᨣ᩠ᨦᩕᩮ᩵​ᩆᩣᩈ᩠ᨶᩣ ᩈ᩠ᩅ᩵ᩁ​ᨲ᩠ᩅᩫ ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ​ᩓ᩠ᩅ​ᨸ᩠ᨶᩮ​ᨤ᩠ᩅᩣ᩠ᨾ​ᨲ᩠ᨶᩨ᩵ᨲᩣᨲ᩠ᨶᩨ᩵ᨧᩱ​ᩀ᩵ᩣ᩠ᨦ​ᨿ᩠ᨦᩥ᩵​ᨴᩦ᩵​ᨯᩱ᩶​ᨡᩮᩢ᩶ᩣ​ᨾᩣ​ᩀᩪ᩵​ᨶᩱ​ᨾᩮᩨᩬᨦ​ᨠᩮᩢ᩵ᩣᨠᩯ᩵​ᩉ᩠ᨦᩯ᩵​ᩉ᩠ᨶᩦ᩶ ᨪ᩠ᨦᩮᩥ᩵​ᨸ᩠ᨶᩮ​ᨷ᩶ᩣ᩠ᨶ​ᨡᩬᨦ​ᨻᩬᩴ᩵ᩋ᩠ᨿᩩ᩶ᨾᩯ᩵ ᩉᩬ᩠᩵ᨾᩁ​ᨡᩬᨦ​ᨣᩕᩥᩈ᩠ᨲᨧᩢᨠᩕ​ᨡᩬᨦ​ᩁᩮᩢᩣ​ᩉ᩠ᩃᩣ᩠ᨿ​ᨤ᩠ᨶᩫ ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ​ᨣ᩠ᨯᩧ​ᨷᩴ᩵​ᨳ᩠ᨦᩮᩥ​ᩅ᩵ᩣ​ᩉ᩠ᨾᩯ​ᨪᩣ᩠ᩅ​ᨸᩦ​ᨲᩬᩴ᩵​ᨾᩣ​ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ​ᨧᩡ​ᨯᩱ᩶​ᨾᩦ​ᩈ᩠ᩅ᩵ᩁ​ᨠᩬᩴ᩵​ᨲ᩠ᨦᩢ᩶ ᨣᩕᩥᩈ᩠ᨲᨧᩢᨠᩕ​ᨷ᩠ᨶᩫ​ᨹ᩠ᨶᩯ᩵ᨯ᩠ᨶᩥ​ᨴᩦ᩵​ᩀ᩠ᨶᩨ​ᩀᩪ᩵​ᨴᩦ᩵​ᩉ᩠ᨶᩦ᩶​

ᨷᩴ᩵​ᨻᩬᩴ​ᨠ᩠ᩃᩱ​ᨧᩣ᩠ᨠ​ᨲ᩠ᩅᩫ​ᨾᩮᩨᩬᨦ​ᨴᩮᩢ᩵ᩣᨯᩱ ᩁᩮᩢᩣ​ᨠᩣ᩠ᨿ​ᩉ᩠ᨾᩪ᩵ᨷ᩶ᩣ᩠ᨶ​ᨴᩦ᩵​ᨷᩴ᩵​ᨾᩦ​ᨤ᩠ᨶᩫ​ᩀᩪ᩵​ᨿᩬ᩶ᩁᩅ᩵ᩣ​ᨾᩦ​ᩈᩮᩨᩬᩋ​ᨡᩮᩢ᩶ᩣ​ᨾᩣ​ᩃᩣ᩠ᨠ​ᨤ᩠ᨶᩫ​ᨻᩱ​ᨠ᩠ᨶᩥ ᩁᩮᩢ᩶ᩣ ᨴ᩠ᨿᩅᨴᩣ᩠ᨦ​ᨧᩣ᩠ᨠ​ᨩ᩠ᨿᨦᩈ᩠ᨶᩯ​ᨾᩣ​ᨯᩱ᩶​ᩈᩬᨦ​ᩅ᩠ᨶᩢ​ᨣᩴ᩵​ᨾᩣ​ᨹ᩠ᩅᩯ​ᨩ᩠ᨿᨦᨡᩬᨦ ᨩ᩠ᨿᨦᨡᩬᨦ​ᨸ᩠ᨶᩮ​ᨾᩮᩨᩬᨦ​ᨡ᩠ᨶᩧ᩶​ᨴᩦ᩵​ᩉ᩠ᨿᩱ᩵​ᨴᩦ᩵​ᩈᩩᨯ​ᨡᩬᨦ​ᨧ᩠ᨦᩢᩉ᩠ᩅᩢᨯ​ᨶ᩵ᩣ᩠ᨶ ᩁᩮᩢ᩶ᩣ ᩀ᩶ᩣ᩠ᨦ​ᨴᩦ᩵​ᩉ᩠ᨶᩦ᩶​ᩈᩬᨦ​ᩅ᩠ᨶᩢ ᨪ᩠ᨦᩮᩥ᩵​ᨸ᩠ᨶᩮ​ᩅᩮᩃᩣ​ᨴᩦ᩵​ᨯᩦ​ᨡ᩠ᨶᩣ᩠ᨯ​ᨸ᩠ᨶᩮ​ᩅᩮᩃᩣ​ᨴᩦ᩵​ᨶ᩵ᩣ​ᩈ᩠ᨶᩫᨧᩱ​ᨠᩢᨷ​ᨯᩱ᩶​ᨠᩣᩴ᩵ᩁᩱ​ᨩᩦᩅ᩠ᨲᩥ ᨡ᩶ᩣᨽᨧᩮᩢ᩶ᩣ​ᨣ᩠ᨿᩮᩥ​ᨸᩡ​ᨧᩮᩢ᩶ᩣ​ᨹᩪ᩶​ᨣᩕᩬᨦ ᨾᩮᩨᩬᨦ​ᩉ᩠ᨶᩦ᩶​ᨾᩣ​ᩓ᩠ᩅ​ᨴᩦ᩵​ᨩ᩠ᨿᨦᩉ᩠ᨾᩱ᩵ ᨹᩬᩴ᩵​ᨡᩮᩢᩣ​ᩈ᩠ᨶᩫᨧᩱ​ᨸ᩠ᨶᩨ​ᩁᩱᨼ᩠ᩃᩮᩥᩢ​ᨴᩦ᩵​ᨿ᩠ᨦᩥ​ᨯᩱ᩶​ᨠᩣᩴ​ᩉ᩠ᩃᩣ᩠ᨿ​ᨶ᩠ᨯᩢ​ᨴᩦ᩵​ᨡ᩶ᩣᨽᨧᩮ᩶ᩣ​ᨿ᩠ᨦᩥ​ᩉᩨ᩶​ᨹᩬᩴ᩵ ᩃᩪᨠ​ᨷ᩵ᩣ᩠ᩅ​ᨡᩬᨦ​ᨡᩮᩢᩣ​ᨣᩴ᩵ ᩀ᩠ᨶᩨ​ᨹᩬᩴ᩵ ᩃᩪᨠ​ᨷ᩵ᩣ᩠ᩅ​ᨡᩬᨦ​ᨡᩮᩢᩣ​ᨤ᩠ᨶᩫ​ᩉ᩠ᨶᩦ᩶​ᩉ᩠ᨾᩯ​ᨷᩴ᩵​ᨾ᩠ᨶᩮᩥ​ᨧᩡ​ᨸ᩠ᨠᩫᨣᩕᩬᨦ​ᨾᩮᩨᩬᨦ​ᩉ᩠ᨶᩦ᩶​ᨴ᩠ᨶᩯ​ᨧᩮᩢ᩶ᩣᨾᩮᩨᩬᨦ ᨪ᩠ᨦᩮᩥ᩵​ᩁᩮᩨᩬ᩵ᨦᩁᩣ᩠ᩅ​ᨡᩬᨦ​ᨡᩮᩢᩣ​ᩁᩮᩢᩣ​ᨧᩡ​ᩉ᩠ᨶᩢ​ᨯᩱ᩶ ᨲᩬᩁ​ᨪᩣ᩠ᩅ​ᨸᩦ​ᨻᩱ​ᨴ᩠ᨦᩢ​ᩉ᩠ᨶ᩶ᩣ​

ᨶᩣᩴ​ᨾᩣ​ᨧᩣ᩠ᨠ​ᨷᩣ᩠ᨦ​ᩈ᩠ᩅ᩵ᩁ​ᨡᩬᨦ​ᩉ᩠ᨶᩢᨦᩈᩨ​ᨷᩕᩅᩢᨲᩥ​ᨩᩦᩅ᩠ᨲᩥ​ᨡᩬᨦ​ᩆᩣᩈ᩠ᨶᩣᨧᩣᩁ᩠ᨿ᩺ ᨯᩢᩬᨠᨲᩮᩥᩬᩁ᩺ ᨯᩣᨶ᩠ᨿᩃ ᨾ᩠ᨣᩯᨣ᩠ᩃᩥᩅᩣᩁᩦ ᨯ᩠ᩅ᩠ᨿᩰᩫ ᨾᩣᨶ᩠ᨻᩫ
