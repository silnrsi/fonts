BJCree - Versions and Changes

Announcement list

If you would like stay informed of updates to BJCree and other SIL fonts, please subscribe to the SIL Font News Announcement List. For more information see About.

Current versions

The latest version of the fonts is always available from the BJCree download page as a .zip archive for all major platforms.

Previous versions

A prior version of the BJCree font remains available from the East Cree language resources page. That version will no longer be updated.

In the future, the BJCree download page will keep previous versions available.

Change history

A detailed list of changes for each version of the fonts is on the Version history page.
