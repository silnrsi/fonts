### 2026-02-11 (SIL WSTech team) BJCree version 7.000
- Released under the SIL Open Font License version 1.1
- Changed to UFO-based workflow with sources available on Github
- Significant improvements to diacritic placement, letter spacing, and line spacing
- Added new characters:
    - U+142E ᐮ an-cans
    - U+142D ᐭ on-cans
    - U+1544 ᕄ leWestCree-cans
    - U+154A ᕊ loWestCree-cans
    - U+154D ᕍ laWestCree-cans
    - U+155F ᕟ theNCree-cans
    - U+1561 ᕡ thiNCree-cans
    - U+1563 ᕣ thiiNCree-cans
- Replaced Latin glyphs with new design based on C059 (New Century Schoolbook) from (URW)++, released under the SIL Open Font License version 1.1.
