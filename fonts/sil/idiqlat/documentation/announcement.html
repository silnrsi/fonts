Idiqlat - Announcement

We are very pleased to announce a new release of Idiqlat, a font for the East Syriac style of the Syriac script.

Changes

This release includes the following changes for this version:

  • Moved the lookup in Contextual Alternates (calt) to Required Contextual Alternates (rclt)

This change allows the font to render the correct forms of kaph, mim, and nun in applications which turn off the Contextual Alternates feature.

Both desktop and web fonts are provided in a single, all-platforms package on the Download page.
