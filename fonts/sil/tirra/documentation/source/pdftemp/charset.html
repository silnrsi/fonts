Tirra - Character Set Support

This font supports over 400 characters from The Unicode Standard. It also includes a number of ligated sequences involving the Labialization Mark.

Note: Inclusion of basic Latin repertoire is provided as a convenience but is not intended for extensive Latin script use. The Latin glyphs are based on the OFL-licensed Source Sans Pro fonts in the Source Sans Github repository.
