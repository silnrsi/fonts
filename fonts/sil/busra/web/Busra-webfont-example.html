Using Busra webfonts

The page demonstrates how to use the WOFF2 version of Busra for web pages, using a local copy of the .woff2 file in this archive. For more information on this technique - and how to use the normal TrueType version as a web font - see Using SIL Fonts on Web Pages.

The body of this document is styled using an external stylesheet, with Busra set as the default font, and loaded directly as a WOFF2 file using the @font-face rule. If you're seeing a sans-serif font here then it is working. Otherwise the text will be in a serif or your browser default. This technique is supported in all modern browsers.

To see specifically how these techniques are specified in the CSS and HTML view the source to this page and the accompanying CSS file.

Busra user-selectable features

The Busra font includes a number of optional features that provide alternative rendering that might be preferable for use in some contexts. The examples below show the results of applying these features. Whether these features are available to users will depend on both the application and the rendering technology being used. The OpenType features are used in other browsers. Further documentation of these features is available in the documentation folder of the font download.

To see specifically how these techniques are specified in the CSS and HTML, view the source to this page and the accompanying CSS file.
