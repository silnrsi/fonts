Akatab - Character Set Support

This font supports over 390 characters from The Unicode Standard as well as a number of Private Use Area (PUA) characters. In total, over 600 glyphs are included, supporting stylistic alternates for a number of characters as well as a large number of ligated sequences.

Note: Inclusion of basic Latin repertoire is provided as a convenience but is not intended for extensive Latin script use. The Latin glyphs are based on the OFL-licensed Source Sans Pro fonts in the Source Sans Github repository.

Private Use Area (PUA) characters

There are a number of private-use characters that are supported in this font. The Private Use Area is a range of codepoints that are reserved in the Unicode for private use by software developers and end users who need a special set of characters for their own purposes.

Tifinagh superscripts

  • U+E300 TIFINAGH SUPERSCRIPT YA
  • U+E301 TIFINAGH SUPERSCRIPT YU
  • U+E302 TIFINAGH SUPERSCRIPT YAY
  • U+E303 TIFINAGH SUPERSCRIPT YI
  • U+E304 TIFINAGH SUPERSCRIPT YO
